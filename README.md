## 🎯 Project Overview

This is a complete, fully responsive website for the **Premier University Computer Club (PUCC)**. The project was designed and developed as part of the **PUCC UI/UX Software Design Hackathon** (July 1, 2026).

The website showcases the club's mission, vision, wings, events, achievements, team, and provides a seamless membership joining experience.

## 🛠️ Technologies Used

### Frontend
- **HTML5** - Semantic markup
- **CSS3** - Custom styles + animations
- **Tailwind CSS** - Utility-first framework
- **JavaScript** - Interactivity, filtering, forms

### Design
- **Figma** - UI/UX design, prototyping
- **Google Fonts** - Manrope + Sora
- **Material Symbols** - Icon library

### Hosting & Deployment
- **Cloudflare Pages** - Live demo
- **GitHub** - Version control

## ✨ Key Features

- ✅ **Fully Responsive** - Desktop, tablet, mobile
- ✅ **Interactive Prototype** - Figma with clickable navigation
- ✅ **Smooth Animations** - Scroll-triggered reveals, typing effect
- ✅ **Event Filtering** - Categorize events by type
- ✅ **Membership Form** - With validation and success state
- ✅ **Modern UI/UX** - Clean, professional, consistent
